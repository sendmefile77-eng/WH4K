# wh4k_myauko — Основной модуль

Структура модуля по Варианту A.

## Формат
- `module.json` — манифест модуля
- `banks/factions/` — банки фракций (Cadian, Blood Angels, Mechanicus, Emperor's Children и т.д.)
- `banks/poses/` — банки поз (в т.ч. фут-фокус)
- `banks/` — остальные банки (characters, locations, atmospheres, bodyDnaTemplates)
- `prompts/` — паки промптов
- `menus/` — меню и направления

## Следующие шаги
1. Заполнить `module.json` реальными данными
2. Создать банки фракций и поз
3. Добавить промпт-паки
4. Подключить модуль в приложение
