# WH4K Roadmap

## v0.1 - Engine skeleton

- Android/Compose project skeleton.
- Core game state models.
- Module manifest and loader contract.
- Prompt assembler v0.
- Atlas Cloud client contract.
- Example module manifest.

## v0.2 - Real module runtime

- Import module folder/zip.
- Validate module layout.
- Persist active module selection.
- Load direction menus from active module.
- Render dynamic hub from module data.

## v0.3 - Visit loop

- Direction selection.
- Showcase screen.
- Act I-V visit runner.
- Frame regeneration.
- Player command input.
- Prompt metadata history.

## v0.4 - Atlas Cloud HTTP client

- API key storage.
- Real generation request.
- Job polling.
- Image cache.
- Error handling and retry.

## v0.5 - Persistence

- Room entities for visits, frames, prompts, generated images, and module metadata.
- DataStore settings for active module, Atlas config, and UI preferences.
